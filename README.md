# Appindependent - Інтернет-магазин побутової техніки

## Опис проєкту
Адаптивний веб-застосунок інтернет-магазину побутової техніки, розроблений в рамках лабораторної роботи №1. Проєкт демонструє використання семантичної HTML-верстки, CSS Flexbox та Grid для побудови сітки карток товарів, а також базової інтерактивності за допомогою JavaScript. 
Інтерфейс є повністю адаптивним і коректно відображається на десктопних, планшетних та мобільних пристроях.

## Файлова структура
- `index.html` — головна сторінка застосунку;
- `/css/style.css` — каскадні таблиці стилів;
- `/js/main.js` — скрипти для роботи інтерактивних елементів (бургер-меню).

## Інструкція із запуску
1. Зклонуйте репозиторій на свій комп'ютер або завантажте його як ZIP-архів.
2. Перейдіть до папки з проєктом.
3. Відкрийте файл `index.html` у будь-якому сучасному веб-браузері (Chrome, Firefox, Safari тощо).
4. *Альтернативний варіант:* відкрийте папку в редакторі Visual Studio Code та запустіть за допомогою розширення Live Server для автоматичного оновлення сторінки.
